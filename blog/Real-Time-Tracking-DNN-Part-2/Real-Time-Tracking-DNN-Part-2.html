论文集《Real-Time Tracking Methods (DNN-Part II)》

Date 三月 30, 2018 Tags Visual Tracking / Deep Neural Networks / Real-Time Tracking

Deep Meta Learning for Real-Time Visual Tracking based on Target-Specific Feature Space (Arxiv)

Janghoon Choi, Junseok Kwon and Kyoung Mu Lee

这篇文章基于SiamFc[1]进行修改。主要目的是为了能够快速的在线更新模板，已达到更好的匹配结果。论文做法是基于Meta-learner思想。具体来说即用Meta-learner来从本身的错误梯度来产生一个对应的权值，将该产生的权值与原有的模板绑定，进行目标的位置预测。感觉比较奇怪的是，即使通过另一种学习机制使得SiamFc框架变得相对复杂，但目前的论文版本来看，结果还不如简单版，感觉应该还有更多可做的空间，例如直接对权值更新，或者利用temporal信息。

UCT: Learning Unified Convolutional Networks for Real-time Visual Tracking(ICCVW2017)

Zheng Zhu, Guan Huang, Wei Zou, Dalong Du and Chang Huang

这篇文章实际上与很早挂在arxiv上的CRT[2]类似(刚才check了一下，发现中了TIP)。基本上可以认为是FCNT[3]的延续。即利用全卷积网络进行目标的跟踪。在预测的过程中，这类全卷积网络实际上就是得到一个response map，在这个map上最大的响应位置即为跟踪目标的位置。当然这篇文章也提出了一些新的做法。第一个就是利用额外的数据进行整个网络的训练。也同时得益与利用额外数据进行的训练，是的feature本身就能很好地区分目标与背景。所以在线跟踪的更新过程中，只迭代了1次就能得到很好的结果。同时，利用全卷积网络的特性，每一次跟踪只需要对当前的状态进行一次网络的forward，所以速度非常的快。第二个提升点即论文提出的训练样本的挑选过程，作者认为完美的样本的response map只能有一个波峰，所以提出了利用response map进行样本挑选的过程。另外，作者在这里也使用了scale filter去估计object的scale来。本文在传统数据例如OTB上表现不错，但在VOT上的表现差强人意。所以有很大的提高的空间。

A Twofold Siamese Network for Real-Time Object Tracking (CVPR2018)

Anfeng He, Chong Luo, Xinmei Tian and Wenjun Zeng

最近关于SiamFc[1]的扩展比较多。这篇文章也算其中的一篇。本文主要claim的点是原始的SiamFc主要是通过大量的数据学习到了Appearance的信息，而关于template的操作则主要是对Appearance进行的匹配。但是，这种仅仅只是基于表关特征并不能很多的跟踪目标，所以本文作者提出了利用pre-trained的网络来提取相关的语意信息，与原始的SiamFc进行融合来提升跟踪的结果。在训练阶段，作者认为两个branch直接在训练时融合会容易出现一个branch训练的很好，而另外一个branch没有得到训练一样。所以作者在论文中采用的是单独训练的策略。即独立训练，仅仅只是在测试的阶段进行融合。另外，作者认为基于pre-trained的网络对于目标跟踪来说有很大的冗余信息，所以作者这里利用attention的尽可能的抑制住不重要的feature channel，同时尽可能的加大重要的feature channel的权值来提升跟踪的结果。

Learning Attentions: Residual Attentional Siamese Network for High Performance Online Visual Tracking (CVPR2018)

Qiang Wang1, Zhu Teng, Junliang Xing, Jin Gao, Weiming Hu1 and Stephen Maybank4

这篇文章也是针对SiamFc[1]的扩展。原始的SiamFc[1]通过大量数据离线训练好feature以及其matching metric部分，在将整体transfer到在线跟踪部分时候，并没有对feature做相应的调整。这也使得SiamFc很难去适应不同target的appearance。为了使得feature部分更好的适应不同的target apparance，本文提出利用attention机制来进行目标的跟踪。具体跟踪的pipeline如上图所示。即在target的feature提取过程中，利用attention去加权不同的spatial以及feature channel。

End-to-end Flow Correlation Tracking with Spatial-temporal Attention (CVPR2018)

Zheng Zhu, Wei Wu, Wei Zou and Junjie Yan

本文实际上也可以看做是对[1,4]进行的拓展。主要是结合帧间信息来做跟踪。本文利用的帧间信息主要是光流。主要的跟踪框架如上图所示。个人感觉本文更像是在利用光流信息来对跟踪中缺乏样本的问题进行改进。例如，本文实际上是利用光流信息来计算出帧与帧之间的点的偏移量，进而利用此偏移量来做warp，最终得到多个不同的样本。在预测跟踪目标的过程中， 同时利用这些warp样本与前一帧的结果来进行目标的定位。而在利用样本的过程中，作者认为这些warp的样本有好有坏，所以作者这里设计了一个temporal attention的网络进行样本的加权。感觉这篇文章结果好，可能是利用到了多帧样本。目前感觉对于VOT的测试结果好可能也是因为通过optical flow来做数据的warp可以带来更多的deformation的样本，使得本身比较适合在VOT上测试，尤其是VOT2017。

Context-aware Deep Feature Compression for High-speed Visual Tracking (CVPR2018)

Jongwon Choi, Hyung Jin Chang, Tobias Fischer, Sangdoo Yun, Kyuewang Lee, Jiyeoup Jeong, Yiannis Demiris and Jin Young Choi

最近基于correlation filter的文章层出不穷，这篇文章实际上也是在CF的框架下，考虑如何降低特征的维度来进行高效并且高速的跟踪。论文作者认为pre-trained的网络对于某些特定的跟踪目标来说，会有大量的冗余。而这种冗余往往取决于特定的跟踪目标以及特定的跟踪场景。所以本文主要的贡献在于利用context信息来进行特征的压缩。具体为：1）将大量图片进行聚类。2）利用聚类后的每一类训练单独的auto-encoder。3）在测试的时候，根据第一帧的信息预测该部分信息属于的聚类中心。4）将利用该类训练的auto-encoder拿出来用来做特征的压缩。5）主题跟踪框架则是传统的CF。感觉有点类似detection to track的过程。只不过这里利用大量的图片学习到与跟踪目标有用的context信息。结果在OTB上表现还可以。当然，为了得到很好地结果，作者在论文中利用了很多小的tricks，具体请参考论文。

Meta-Tracker: Fast and Robust Online Adaptation for Visual Object Trackers (ECCV2018)

最近基于meta-learning来做各种应用的层出不穷。感觉meta-learning的潜在思想是利用大数据把该记住该模拟的都记住，例如学习率。应用阶段则是通过将该memorize住的东西应用到对应阶段。这篇文章也是从该思想出发，利用大数据学习到重要的初始化以及在线更新的信息，i.e., 初始化参数以及在线更新学习率。个人猜测这种通过学习meta参数的模式比较依赖学习的方式。比如这篇文章中，对于初始化参数以及学习率的学习，并不仅仅只是考虑当前帧样本，同时将未来某帧的信息考虑进来。这实际上等同于告诉学习机：你现在的feature在某帧之后的表达欠缺，需要通过这样的学习参数才能更好地做到调整，使得在某帧的表达合适。借由此分析，是否现阶段SiamFc[1]这种框架结果好，也是因为记住了更多的未来帧信息呢（暂未验证）。

Learning Dynamic Memory Networks for Object Tracking (ECCV2018)

这篇文章如他自己题目一样，主要是利用template matching去做tracking，然后收集不同时间段下的template作为memory。在利用LSTM去做类似于template的挑选机制。整个过程只有feed-forward，所以结果是速度还比较快。但其最终跟踪的结果一般，应该是在做memory挑选的过程中，仅仅利用LSTM不足以做到足够好，后续应该还有很大的提升空间。
